 
-- SUMMARY --

Dartik is a dark version and a sub-theme of Bartik - Drupal 7 core's default theme.
It is basically an override of all color styles from Bartik leaving all other styles untouched, 
to be inherited from the original theme.

For a full description of the theme, visit the project page:
  http://drupal.org/project/dartik

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/dartik



-- REQUIREMENTS --
   
* Bartik (core) - Doesn't have to be enabled.



-- INSTALLATION --

 Install as usual, see http://drupal.org/node/176045 for further information.



-- CONFIGURATION --

* Configure theme settings in Administration » Appearance » Settings » Dartik  :

  -  Color theme (require color module from core)

  - Toggle theme items display - primary links, logo etc.

  - Logo image settings

  - Shortcut icon (favicon) settings



-- CUSTOMIZATION --

* Like with any other drupal theme you are able to customize just about every 
bit of this theme.

- drupal.org theme guide is here : http://drupal.org/documentation/theme

- it is recomended to create a sub-theme of your own to put all your overrides in :
  Creating a sub-theme - http://drupal.org/node/225125
